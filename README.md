# CleanPlanet

Учебный программный проект, реализующий информационную систему учёта партнёров, услуг и расчёта себестоимости услуг химчистки.  
Проект включает реляционную базу данных и настольное приложение на платформе Windows Forms.

---

## Состав проекта

- `src/CleanPlanet.WinForms/` — исходный код Windows Forms приложения  
- `db/CleanPlanet.sql` — SQL-скрипт создания и наполнения базы данных  

---

## Используемые технологии

- СУБД: Microsoft SQL Server
- Язык программирования: C#
- Платформа: .NET (Windows Forms)
- Доступ к данным: Entity Framework Core
- Среда разработки: Visual Studio

---

## Развёртывание базы данных

1. Открыть Microsoft SQL Server Management Studio.
2. Выполнить SQL-скрипт `db/CleanPlanet.sql`.
3. Убедиться, что база данных `CleanPlanet` создана и содержит таблицы и тестовые данные.

---

## Запуск приложения

1. Открыть решение `CleanPlanet.WinForms` в Visual Studio.
2. Проверить строку подключения к SQL Server в проекте.
3. Собрать и запустить приложение.

---

## Назначение приложения

Приложение предназначено для:
- просмотра и редактирования данных о партнёрах;
- анализа истории оказанных услуг;
- расчёта потребности в материалах и себестоимости услуг на основе нормативов.
